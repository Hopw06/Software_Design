This is an assignment from instructor:

Modeling a home entertainment system and the ability for a user to be able to use a single remote control to control multiple digital devices
such as a TV or sound system etc.

+ Create a RemoteControl class with three method:
    - A method to choose the device to control.
    - A method to turn on.
    - A method to turn off.

+ Create class for some devices: Projector, TV, SurroundSoundSystem

+ Create Person class, this will be user of the home entertainment system.

Apply open closed principle.

Add new device to system.
